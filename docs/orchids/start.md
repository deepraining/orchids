
## orchids.start

Start application.

```
orchids.start(pageName, data)
```

### pageName

* `desc`: The page name defined before.
* `type`: `string`
* `more`: see [orchids.start](./start_page.md)

### data

* `desc`: Parameter to be used by next Page's `onCreate` method.
* `more`: see [orchids.start](./start_page.md)
